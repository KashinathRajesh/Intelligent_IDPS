# Intelligent Intrusion Detection & Prevention System (IDPS)

## Overview
A Python-based IDPS that detects network anomalies using signature-based rules and machine learning, with active response capabilities.

## Features
- Packet Sniffing (Scapy)
- Signature-based Detection
- Anomaly Detection (Isolation Forest)
- Automated Blocking (IPS)
- Web Dashboard (Flask)
